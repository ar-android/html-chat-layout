# Simple HTML Chat Layout

This is simple project created using [Tailwind CSS](https://tailwindcss.com).

To get started, clone the project and install the dependencies:

```
# Using npm
npm install

# Using Yarn
yarn
```

After that, start up Webpack:

```
webpack --watch
```
